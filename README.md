# java-web-sbt-spring-thyme-keystore-hello-world

## Description
A springboot web app with thyme and self-signed ssl support.

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`
Available at https://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
